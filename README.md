This addon lets the user check for the five adds needed for the ICC achievement "Full House" https://www.wowhead.com/achievement=4611/full-house-25-player#. It records if each add was seen and how long ago it was seen. It also changes if it fails to detect an add that was found before because that add died.

# Commands

/fhh show: Toggles the shown/hidden of the frame.

/fhh resetpos: Resets the frame to the center of the screen.
